# Lessons 8–9: Mid-Unit 3 Assessment: Write an Argument Essay
**Grade 6: Module 4: Unit 3: Lessons 8–9**

**Summary:** In the first half of this unit, students have analyzed a model, written collaboratively, and planned their own independent argument essay. This lesson assesses those skills as students use their plans to compose their argument essays about the focus figure they researched.

**Focus Standards:** RI.6.1, RI.6.3, W.6.1, W.6.4, W.6.5, W.6.6, W.6.7, W.6.8, W.6.9b, L.6.2, L.6.3, L.6.6

**Learning Targets:**
- I can write an argument essay about why my focus figure’s accomplishments are remarkable. (RI.6.1, RI.6.3, RI.6.10, W.6.1, W.6.4, W.6.5, W.6.6, W.6.7, W.6.8, W.6.9b, W.6.10, L.6.2, L.6.3, L.6.6)

**Agenda:** Opening → Engage the Learner – W.6.10 (10 minutes) → Work Time → Mid-Unit 3 Assessment: Write an Argument Essay (65 minutes) → Closing and Assessment → Track Progress – W.6.1 (15 minutes)
