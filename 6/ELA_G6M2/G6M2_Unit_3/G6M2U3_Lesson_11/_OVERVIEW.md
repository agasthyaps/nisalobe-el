# Lesson 11: Prepare for Performance Task: Analyze a Model
**Grade 6: Module 2: Unit 3: Lesson 11**

**Summary:** Students begin working on the Module 2 Performance Task by analyzing a model and planning their visual for the Solution Symposium. Students also share their progress on their independent research reading task.

**Focus Standards:** SL.6.2, SL.6.5

**Learning Targets:**
- I can analyze a model for characteristics of an effective performance task presentation. (SL.6.2)
- I can create an effective performance task presentation using visuals. (SL.6.5)

**Agenda:** Opening → Engage the Learner – W.6.10 (5 minutes) → Work Time → Analyze Model Performance Task – SL.6.2 (15 minutes) → Prepare Performance Task Visual – SL.6.5 (15 minutes) → Closing and Assessment
