# Lesson 10: Prepare for Performance Task: Analyze a Model and Select a Text
**Grade 6: Module 3: Unit 3: Lesson 10**

**Summary:** Students prepare for the end of unit assessment and performance task by analyzing a model and then reviewing and choosing a text with which to work.

**Focus Standards:** RI.6.1, RI.6.6, SL.6.2

**Learning Targets:**
- I can paraphrase the key ideas and demonstrate understanding of the perspective being conveyed in the model performance task recording. (SL.6.2)
- I can analyze a model for characteristics of an effective performance task.

**Agenda:** Opening → Engage the Learner (5 minutes) → Work Time → Analyze a Model Recording – SL.6.2 (20 minutes) → Select a Text – RL.6.10, RI.6.10 (15 minutes) → Closing and Assessment
