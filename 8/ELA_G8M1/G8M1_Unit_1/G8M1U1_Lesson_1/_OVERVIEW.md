# Lesson 1: Discover Our Topic: *Summer of the Mariposas*
**Grade 8: Module 1: Unit 1: Lesson 1**

**Summary:** In this lesson, students observe multiple artifacts to infer the module topic, Latin American Folklore. They also create an anchor chart for Work to Become Ethical People.

**Focus Standards:** RL.8.1, RI.8.1, SL.8.1

**Learning Targets:**
- I can infer the topic of this module from the resources. (RL.8.1, RI.8.1)

**Agenda:** Opening → Engage the Learner – SL.8.1 (5 minutes) → Work Time → Infer the Topic – RL.8.1 (10 minutes) → Introduce the Performance Task and Module Guiding Questions (10 minutes) → Launch the Text: *Summer of the Mariposas* (10 minutes)
