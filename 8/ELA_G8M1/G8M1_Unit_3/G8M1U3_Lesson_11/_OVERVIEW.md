# Lesson 11: Plan End of Unit 3 Assessment Essay
**Grade 8: Module 1: Unit 3: Lesson 11**

**Summary:** In this lesson, students will briefly analyze a new Painted Essay® model and create a plan for their end of unit assessment essay using a graphic organizer.

**Focus Standards:** W.8.5

**Learning Targets:**
- I can plan an informative essay that has points that are supported by facts and details. (W.8.2, W.8.5)
- I can give kind, helpful, and specific feedback to my partner. (W.8.5, SL.8.1)

**Agenda:** Opening → Engage the Learner – W.8.5 (5 minutes) → Work Time → Guided Practice: Plan End of Unit 3 Assessment Essay – W.8.5 (15 minutes) → Independent Practice: Plan End of Unit 3 Assessment Essay – W.8.5 (10 minutes) → Peer Critique: End of Unit 3 Assessment Writing Plan – W.8.5 (10 minutes)
