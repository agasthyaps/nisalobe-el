# Lesson 2: Mid-Unit 3 Assessment: Determine a Central Idea
**Grade 8: Module 1: Unit 3: Lesson 2**

**Summary:** In this lesson, students read an essay and identify main ideas and key details for the mid-unit assessment.

**Focus Standards:** RI.8.1, RI.8.2, RI.8.4, L.8.4a, L.8.4c, L.8.4d

**Learning Targets:**
- I can determine a central idea and how it is conveyed through details in an informational text. (RI.8.1, RI.8.2)

**Agenda:** Opening → Return End of Unit 2 Assessments (5 minutes) → Engage the Learner (5 minutes) → Work Time → Mid-Unit 3 Assessment: Determine a Central Idea – RI.8.2 (20 minutes) → Closing and Assessment
