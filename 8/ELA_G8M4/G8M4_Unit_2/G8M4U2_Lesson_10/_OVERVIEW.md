# Lesson 10: Write a Literary Argument Essay: Analyze a Model
**Grade 8: Module 4: Unit 2: Lesson 10**

**Summary:** In this lesson, students analyze a model literary argument essay in order to determine criteria to apply to their own writing.

**Focus Standards:** W.8.1, W.8.4

**Learning Targets:**
- I can find the gist of a model essay. (RI.8.1)
- I can identify the parts of a model argument essay and explain the purpose of each. (W.8.1)
- I can generate criteria for an effective argument essay. (W.8.1, W.8.4)

**Agenda:** Opening → Engage the Learner – W.8.1 (5 minutes) → Work Time → Find the Gist of a Model Argument Essay (10 minutes) → Analyze a Model Argument Essay – W.8.1 (15 minutes) → Argument Writing Checklist – W.8.1, W.8.4 (10 minutes)
