# Lesson 4: Write an Argument Essay: Analyze a Model
**Grade 8: Module 2: Unit 3: Lesson 4**

**Summary:** In this lesson, students analyze a model argument essay to prepare to write their own.

**Focus Standards:** RI.8.1, W.8.1, W.8.4

**Learning Targets:**
- I can find the gist of a model essay. (RI.8.1)
- I can identify the parts of a model argument essay and explain the purpose of each. (W.8.1)
- I can generate criteria for an effective argument essay. (W.8.1, W.8.4)

**Agenda:** Opening → Engage the Learner – W.8.1 (5 minutes) → Work Time → Find the Gist of a Model Argument Essay (10 minutes) → Analyze a Model Argument Essay – W.8.1 (15 minutes) → Argument Writing Checklist – W.8.1 (10 minutes)
