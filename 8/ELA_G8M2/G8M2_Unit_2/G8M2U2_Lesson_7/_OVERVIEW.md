# Lesson 7: Write an Informative Essay: Analyze a Model
**Grade 8: Module 2: Unit 2: Lesson 7**

**Summary:** In this lesson, students analyze a model essay and analyze how it meets the criteria for an informative essay.

**Focus Standards:** RI.8.1, W.8.2, W.8.4

**Learning Targets:**
- I can find the gist of a model essay.
- I can analyze a model essay to generate criteria of an effective informative essay on my selected research case study. (W.8.2, W.8.4)

**Agenda:** Opening → Engage the Learner – W.8.2 (5 minutes) → Work Time → Read for Gist: Model Essay – “GMOs” (15 minutes) → Analyze a Model Essay – W.8.4 (15 minutes) → Closing and Assessment
