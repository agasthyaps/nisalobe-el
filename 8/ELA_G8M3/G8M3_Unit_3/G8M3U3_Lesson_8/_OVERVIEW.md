# Lesson 8: Plan a Narrative: Create a Fictional Upstander Profile
**Grade 8: Module 3: Unit 3: Lesson 8**

**Summary:** Students create a profile of their fictional upstander.

**Focus Standards:** W.8.3a, W.8.4, W.8.5, SL.8.1

**Learning Targets:**
- I can create a fictional upstander profile, including setting, character information, and critical life experiences. (W.8.3a)
- I can orient the reader by introducing the narrator, character, and setting of my upstander’s interview. (W.8.3a)
- I can give kind, specific, and helpful feedback to my peers.

**Agenda:** Opening → Engage the Learner – W.8.3 (5 minutes) → Work Time → Create an Upstander Profile – W.8.3a (15 minutes) → Profile Feedback: Praise, Question, Suggestion – W.8.5 (15 minutes) → Closing and Assessment
