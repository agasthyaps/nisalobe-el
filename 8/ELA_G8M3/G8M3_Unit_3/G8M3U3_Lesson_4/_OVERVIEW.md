# Lesson 4: Voices of Upstanders: Miep Gies
**Grade 8: Module 3: Unit 3: Lesson 4**

**Summary:** Students read an informational text about a Holocaust upstander, consider characteristics that make this person an upstander, and write a reflection.

**Focus Standards:** RI.8.1, RI.8.2, W.8.4, L.8.2a, L.8.2b

**Learning Targets:**
- I can determine a central idea and analyze its development in “Miep Gies.” (RI.8.2)
- I can write a text reflection about “Miep Gies.” (RI.8.1, RI.8.2)

**Agenda:** Opening → Engage the Learner – L.8.2b (5 minutes) → Work Time → Say Something: Read Text and Identify Central Idea – RI.8.2 (20 minutes) → Write a Text Reflection – RI.8.1 (15 minutes) → Closing and Assessment
