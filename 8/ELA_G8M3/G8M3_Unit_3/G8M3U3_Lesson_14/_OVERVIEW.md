# Lesson 14: Performance Task: Create Prompt Cards
**Grade 8: Module 3: Unit 3: Lesson 14**

**Summary:** Students create prompt cards to prepare for their performance task presentation.

**Focus Standards:** W.8.4, W.8.5, W.8.10

**Learning Targets:**
- I can draft a reflection on the planning of my graphic panels. (W.8.4)
- I can identify key points from my reflection to create prompt cards for my presentation. (W.8.10)
- I can revise and strengthen my prompt cards using peer feedback. (W.8.5)

**Agenda:** Opening → Engage the Learner – SL.8.5 (5 minutes) → Work Time → Draft a Reflection – W.8.4 (15 minutes) → Draft Key Points for a Presentation – W.8.10 (10 minutes) → Closing and Assessment
