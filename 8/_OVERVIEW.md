## Structure
```
8/
  ELA_G{grade}M{n}/          ← module (topic-based unit of study)
    G{grade}M{n}_Unit_{u}/    ← unit (10–16 lessons)
      G{grade}M{n}U{u}_Lesson_{l}/  ← individual lesson
        _OVERVIEW.md          ← LLM summary
        *-ModuleLessons.md    ← full lesson plan
        Student_Materials/    ← student-facing docs
        Teacher_Materials/    ← teacher-facing docs
    G{grade}M{n}_Performance_Task/  ← final assessment materials
```
