# Lesson 8: Write an Argument Essay: Draft an Introduction
**Grade 7: Module 4: Unit 2: Lesson 8**

**Summary:** Using the Painted Essay® structure, students sort and color code parts of the model argument essay introduction. Then they participate in a Language Dive to analyze what makes the main claim of the model essay effective. Students then work with a partner to draft an introduction to their pair argume

**Focus Standards:** W.7.1a, L.7.1c

**Learning Targets:**
- I can write an introduction for my essay giving context on plastic pollution, acknowledging an opposing claim, and clearly stating the main claim of the piece. (W.7.1a)

**Agenda:** Opening → Engage the Learner – L.7.1c (5 minutes) → Work Time → The Painted Essay®: Sort and Color-Code the Parts of an Introduction – W.7.1a (15 minutes) → Language Dive: Model Essay, Main Claim – W.7.1a (10 minutes) → Closing and Assessment
