# Lesson 7: Write an Argument Essay: Analyze a Model
**Grade 7: Module 4: Unit 2: Lesson 7**

**Summary:** Students use the Painted Essay® structure to analyze a model argument essay about the beginning of the plastic life cycle. Then students work in pairs to plan an argument essay about the end of the plastic life cycle being the best place to reduce plastic pollution.

**Focus Standards:** W.7.1, W.7.5

**Learning Targets:**
- I can identify the parts of a model argument essay and explain the purpose of each. (W.7.1, W.7.4)
- I can plan an argument essay to support the claim that we should reduce plastic pollution by targeting the end of the plastic life cycle. (W.7.4, W.7.5)

**Agenda:** Opening → Engage the Learner – W.7.1 (5 minutes) → Work Time → Analyze a Model – W.7.1 (25 minutes) → Closing and Assessment → Pair Practice: Plan an Argument Essay – W.7.5 (15 minutes)
