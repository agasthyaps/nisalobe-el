# Lessons 6–7: Mid-Unit 3 Assessment: Revise and Edit a Documentary Script
**Grade 7: Module 4: Unit 3: Lessons 6–7**

**Summary:** Students participate in a Language Dive on a sentence from the Model Documentary Script, analyzing how the author uses evidence to support the argument. Then students engage in a Tuning protocol as a triad to give each other feedback on the effective use of claims and findings (including points, key

**Focus Standards:** W.7.5, SL.7.4

**Learning Targets:**
- I can revise and strengthen a documentary script, focusing on how well purpose and audience have been addressed. (W.7.5)

**Agenda:** Opening → Engage the Learner (5 minutes) → Work Time → Language Dive: Model Documentary Script, Argument Section – SL.7.4 (15 minutes) → Tuning Protocol – W.7.5 (30 minutes) → Closing and Assessment
