# Lesson 5: Write a Narrative: Plan Character and Setting
**Grade 7: Module 1: Unit 3: Lesson 5**

**Summary:** In this lesson, students finish reading *A Long Walk to Water* by analyzing the author’s note. Then students use the model narrative *Nasreen’s Secret School* to plan the character and setting(s) of their own narratives about a Lost Child of Sudan. Students conclude the lesson by sharing their plans

**Focus Standards:** W.7.3a, W.7.5

**Learning Targets:**
- I can create a character profile for my narrative. (W.7.3a, W.7.5)
- I can create accurate settings based on the places I have read about. (W.7.3a, W.7.5)

**Agenda:** Opening → Engage the Learner – W.7.3a (5 minutes) → Work Time → Read *A Long Walk to Water*, “Message from Salva” and Author’s Note – W.7.5 (15 minutes) → Introduce Work to Contribute to a Better World – SL.7.1 (5 minutes) → Plan Character and Setting – W.7.3a (15 minutes)
