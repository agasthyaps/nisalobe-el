Read this learning target:

- I can create accurate settings based on the places I have read about.

Why is it important to create accurate settings for this narrative? Do all narratives need to have settings based on real life?

Because the story is based on real events, the setting should be accurate. But if the story weren’t based on real events, the settings wouldn’t have to be realistic.
