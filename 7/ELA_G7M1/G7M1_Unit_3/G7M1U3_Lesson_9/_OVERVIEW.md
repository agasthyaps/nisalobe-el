# Lesson 9: Introduce the Performance Task
**Grade 7: Module 1: Unit 3: Lesson 9**

**Summary:** In this lesson, students review the performance task assignment and anchor chart to generate criteria of an effective performance task. Students also review the author’s notes from *A Long Walk to Water* and *Nasreen’s Secret School* to use them as models for the author’s note to their own ebooks. T

**Focus Standards:** W.7.2, W.7.4, W.7.5, W.7.6

**Learning Targets:**
- I can generate criteria for an effective ebook. (W.7.4, W.7.5)
- I can use technology to create the author’s note for an ebook. (W.7.2, W.7.6)

**Agenda:** Opening → Engage the Learner – RI.7.6 (5 minutes) → Work Time → Introduce the Performance Task – W.7.4 (15 minutes) → Write Author’s Note – W.7.2 (20 minutes) → Closing and Assessment
