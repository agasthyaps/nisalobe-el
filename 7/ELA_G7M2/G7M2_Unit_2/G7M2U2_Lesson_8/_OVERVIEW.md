# Lesson 8: Write an Informative Essay: Analyze a Model
**Grade 7: Module 2: Unit 2: Lesson 8**

**Summary:** Students analyze a model informative essay using the Painted Essay® structure. Then they work in pairs to plan their own peer informative essay.

**Focus Standards:** W.7.2, W.7.4, W.7.5

**Learning Targets:**
- I can explore a model and determine criteria of an informative essay. (W.7.2)
- I can use the Painted Essay® structure to analyze a model. (W.7.2, W.7.4)
- I can plan an informative essay, focusing on task, purpose, and audience. (W.7.4, W.7.5)

**Agenda:** Opening → Engage the Learner – W.7.2 (5 minutes) → Work Time → Analyze a Model – W.7.2 (20 minutes) → Closing and Assessment → Pair Practice: Plan an Informative Essay – W.7.5 (20 minutes)
