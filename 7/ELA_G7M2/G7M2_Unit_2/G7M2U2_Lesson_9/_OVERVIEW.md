# Lesson 9: Write an Informative Essay: Draft an Introduction
**Grade 7: Module 2: Unit 2: Lesson 9**

**Summary:** Students use the Painted Essay® structure to analyze the introduction of the Model Informative Essay. Then they participate in a Language Dive on the focus statement of the Model Informative Essay. Students conclude the lesson by drafting the introduction to their peer essay.

**Focus Standards:** W.7.2a, L.7.1a

**Learning Targets:**
- I can write an introduction for my essay giving context on crime detection and epidemiology, clearly stating the focus of the piece. (W.7.2a)

**Agenda:** Opening → Engage the Learner – W.7.2a (5 minutes) → Work Time → The Painted Essay®: Sort and Color-Code the Parts of an Introduction – W.7.2a (15 minutes) → Language Dive: Model Essay, Focus Statement – W.7.2a, L.7.1a (10 minutes) → Closing and Assessment
